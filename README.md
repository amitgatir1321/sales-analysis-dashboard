# 📊 Sales Analysis Dashboard

> An interactive Power BI dashboard designed to analyze sales performance and support data-driven decision-making.

---

## 🔍 Overview
This project presents an interactive Sales Analysis Dashboard built using Power BI. It provides insights into sales performance across different cities, products, and payment methods.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing cities and products  
- Track monthly revenue trends  

---

## 🛠 Tools & Technologies
- Power BI  
- Microsoft Excel  
- Data Visualization Techniques  

---

## 📊 Dashboard Preview

![Sales Dashboard](dashboard.png)

---

## 📈 Key Insights
- Sales show fluctuations across months, with peak performance around mid-year  
- Certain cities contribute significantly higher revenue than others  
- Revenue distribution across payment methods is relatively balanced  
- Multiple products contribute to revenue, indicating diversified sales  

---

## 📊 Features
- Interactive slicer for filtering data by city  
- Monthly sales trend visualization (Line Chart)  
- City-wise revenue comparison (Bar Chart)  
- Payment method distribution (Pie Chart)  
- Product performance analysis (Treemap)  
- KPI cards for quick insights (Total Revenue & Units Sold)
